# Jadoo - Travel Booking Landing Page

A responsive, modern travel landing page built with **HTML, CSS, Bootstrap 5, Vite, and Sass**.  
It includes sections like Hero, Features, Testimonials, Logo Slider, and Newsletter CTA.

## Features

- Fully responsive design
- Bootstrap 5 components
- Swiper.js slider for testimonials and logos
- Sass for styling and custom animations
- SEO-friendly structure

## Tech Stack

- HTML5 & CSS3
- Bootstrap 5
- Vite
- Sass
- Font Awesome

- Swiper.js
